As we have said before, in the Vim text editor we do not have a mouse, the things that you would normally do with the mouse in any other text editor you cannot do.

To add grace to the matter, in a terminal (where we normally run Vim) we don't have any kind of button or anything like that. All the possible icons that we see are made with text.

This is why we have modes in Vim, as we cannot do everything we would normally do with a keyboard and a mouse with just a keyboard, we have to divide the editor into modes (or layers), so that the same keys have functions different

Normal mode

If you open a file in Vim, as we explained in the third chapter, you'll notice that you can't write anything in the file.

If you press keys you will end up being able to write, even though you don't really know how you did it.

This happens because when starting Vim, the default mode is normal mode or command mode. This mode is used to view the file we want to open, move through it, and execute Vim commands.

In the next chapter we will explain how to move inside a file in Vim, but for the moment you can move the cursor in the file using the arrows on your keyboard.

To change modes, we have to use the keys on our keyboard.

Insert mode

We use insert mode to write text to a file.

To enter insert mode press the "i" key from normal mode

You will see that you can now type with the cursor as you normally do.

You can check the current mode you are in by reading the text in the bar at the bottom left of your terminal. In insert mode, this bar will say something like "--INSERT--"

To return to normal mode, from where you can execute commands, use the escape key, if you press it twice quickly, you will return to normal mode faster.

To save the changes to a file in Vim, enter the command bar and type w, that is :w

At the end of chapter 3 we already explained how to exit Vim, but we will expand on that information. Use :q to exit Vim.

If you want to exit saving changes to the file, use :w and then :q.

To avoid having to type two different commands, you can simplify this in :wq

Y si quieres salir de Vim sin guardar los cambios, utiliza :q!.
