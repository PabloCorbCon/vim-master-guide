Ok, so you have decided that you want to start using Vim, as we said in the first chapter of this guide, one of the characteristics of Vim is that it can run almost in every system, in fact it's normally pre-installed in almost all Unix systems. In this chapter we will cover how to install Vim in all those systems, so let's begin!

Linux

Installing in Ubuntu / Debian

The easiest way of installing Vim in Ubuntu based distributions or in Debian is using Apt

sudo apt-get update && sudo apt-get install vim

Installing in Arch based distributions

If you need more information about installing Vim in arch linux, you can check this article of the Arch wiki page.

You should first update pacman using pacman -Syy and then install vim using: pacman -S vim

Installing Vim in fedora based distributions

To install Vim in Fedora based distributions, we will use dnf, So before installing anything, let's update dnf:

sudo dnf update

Then, type the following command:

sudo dnf install vim-enhanced

Installing from source

You can install Vim from it's GitHub repository, simply clone the repository into your local computer

git clone https://github.com/vim/vim.git

Please, check the repository itself for more information about it.

Windows

To install Vim in Windows, please check the offitial Vim page, there you'll find all the executables for installing Vim in Windows.
