There are two really usefull commands you need to know.

:vs

and

:hs

:vs (vertical split) is used to open a file in another window (but inside the same terminal). It will open the file vertically.

:hs does the same thing, but it splits the new file horizontally.

For using :vs or :hs type:

:vs / :hs route-to-the-file

For example:

:hs templates/index.html {% endblock post %}
